# NGS vs TGS Slides

对比二代测序（NGS）与三代测序（TGS）的教学幻灯，纯静态页面，包含脚本与样式文件。

## 快速开始

- 直接打开 `index.html`（建议使用 Chrome/Edge）
- 或使用本地静态服务器（推荐）：
  ```bash
  npm run dev   # 使用 npx serve 在本目录启动，默认 http://localhost:5175
  ```

## 目录结构

```
slides/ngs-vs-tgs/
├─ index.html
├─ styles.css
├─ script.js
└─ README.md
```
