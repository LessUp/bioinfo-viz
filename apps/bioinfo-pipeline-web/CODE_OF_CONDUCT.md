# 行为准则

本项目遵循 Contributor Covenant 的原则，旨在为所有参与者提供一个开放、包容与安全的社区。

## 我们的承诺

- 尊重每一位贡献者与使用者。
- 接纳不同观点与经验。
- 建设性地沟通与协作。

## 我们的标准

积极行为示例：

- 使用友善且包容的语言
- 尊重不同观点
- 乐于接受建设性建议

不可接受行为示例：

- 人身攻击、歧视、骚扰
- 公开或私下的侮辱
- 发布他人隐私信息

## 责任与执行

维护者有权删除或拒绝与本准则不一致的评论、提交、代码、Wiki 编辑、问题与其他贡献，并可临时或永久封禁违规者。

## 联系

如有问题或需要举报，请在 GitHub 上联系维护者，或通过 SECURITY.md 中的安全联络方式。
